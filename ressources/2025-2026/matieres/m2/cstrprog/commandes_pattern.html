2. Création du modèle

3. Variables Choco

3.1 IntVar

Domaine par intervalle 

IntVar x = model.intVar("X", 1, 10);

Signifie :

X ∈ [1,10]

Domaine explicite 

IntVar x =
    model.intVar(
        "X",
        new int[]{1,3,7}
    );

Signifie :

X ∈ {1,3,7}

Variable constante 

IntVar c = model.intVar(5);

3.2 BoolVar 

BoolVar b = model.boolVar("B");

Valeurs :

0 ou 1

3.3 Tableau de variables

IntVar[] 

IntVar[] vars =
    model.intVarArray(
        10,
        1,
        20
    );

BoolVar[] 

BoolVar[] vars =
    model.boolVarArray(10);

3.4 Matrice de variables 

IntVar[][] grid =

6. Contraintes arithmétiques

6.1 Égalité 

model.arithm(x, "=", y).post();

6.2 Différence 

model.arithm(x, "!=", y).post();

6.3 Inférieur 

model.arithm(x, "<", y).post();

6.4 Supérieur 

model.arithm(x, ">", y).post();

6.5 Comparaison avec constante 

model.arithm(x, "<=", 10).post();

6.6 Forme avancée 

model.arithm(
    x,
    "+",
    4,
    "<=",
    y
).post();

Signifie :

x + 4 <= y

7. Contraintes globales

7.1 AllDifferent 

model.allDifferent(vars).post();

7.2 Sum() 

model.sum(
    vars,
    "=",
    10
).post();

Signifie :

x1 + x2 + ... = 10

7.3 Scalar()

Très important.

model.scalar(
    vars,
    coeffs,
    "=",
    value
).post();

Exemple 

model.scalar(
    new IntVar[]{x,y,z},
    new int[]{100,50,20},
    "=",
    240
).post();

Signifie :

100x + 50y + 20z = 240

7.4 Times() 

model.times(
    x,
    y,
    z
).post();

Signifie :

x * y = z

7.5 Max() 

model.max(
    maxVar,
    vars
).post();

Signifie :

maxVar = max(vars)

7.6 OR logique 

model.or(
    c1,
    c2,
    c3
).post();

7.7 AND logique 

model.and(
    c1,
    c2
).post();

8. Contraintes sur une grille

8.1 Contraintes lignes 

for(int i=0;i<n;i++){

    model.sum(
        grid[i],
        "=",
        value
    ).post();
}

8.2 Contraintes colonnes 

for(int j=0;j<n;j++){

    IntVar[] col =
        new IntVar[n];

    for(int i=0;i<n;i++){

        col[i] = grid[i][j];
    }

    model.sum(
        col,
        "=",
        value
    ).post();
}

8.3 AllDifferent sur grille

Flatten 

IntVar[] flat =
    new IntVar[n*n];

int k = 0;

for(int i=0;i<n;i++){

    for(int j=0;j<n;j++){

        flat[k++] =
            grid[i][j];
    }
}

Contrainte 

model.allDifferent(flat).post();

15. Concepts fondamentaux

15.1 IntVar ≠ int

Très important.

Java classique 

int x = 5;

Valeur connue immédiatement.

CSP 

IntVar x;

Valeur inconnue :

  • propagation,
  • recherche,
  • résolution.

15.2 Ne jamais faire 

x + y

si :

x et y sont des IntVar

15.3 Toujours utiliser

  • sum()
  • scalar()
  • arithm()
  • times()

16. Arc Consistency vs Bound Consistency

20. Pièges classiques

Oublier .post() 

model.sum(...).post();

Confondre IntVar et int

Utiliser allDifferent sur matrice 2D

Toujours flatten avant.

Oublier les contraintes de domaine

Exemple packing :

x + width <= boardWidth

Mauvaise modélisation

Toujours chercher :

moins de variables
plus petits domaines
contraintes globales
