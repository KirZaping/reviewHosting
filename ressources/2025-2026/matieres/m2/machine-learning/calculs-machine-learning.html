1. Min-Max Scaling
Le Min-Max scaling sert à remettre des données sur une même échelle.

Exemple : si on compare une taille en cm et un revenu en euros, le revenu a des valeurs beaucoup plus grandes. Certains algorithmes peuvent alors donner trop d’importance au revenu. Le scaling permet de ramener les valeurs entre 0 et 1.

Formule générale :

x_scaled = ((x - min(x)) / (max(x) - min(x))) * (max_new - min_new) + min_new

Pour mettre entre 0 et 1 :

min_new = 0
max_new = 1

Donc la formule devient :

x_scaled = (x - min) / (max - min)

Exemple avec les tailles :

Height = [160, 175, 180, 190]

Minimum :

min = 160

Maximum :

max = 190

Pour 175 :

x_scaled = (175 - 160) / (190 - 160)
x_scaled = 15 / 30
x_scaled = 0.5

Donc 175 devient 0.5.

Résultat :

Height Scaled [0, 1]
160 0
175 0.5
180 0.6667
190 1

À retenir :

La plus petite valeur devient 0.
La plus grande valeur devient 1.
Les autres valeurs sont entre 0 et 1.

2. Min-Max Scaling Entre -1 Et 1
On utilise la même formule, mais on change le nouvel intervalle.

Ici :

min_new = -1
max_new = 1

Formule :

x_scaled = ((x - min) / (max - min)) * (1 - (-1)) + (-1)

Comme :

1 - (-1) = 2

La formule devient :

x_scaled = ((x - min) / (max - min)) * 2 - 1

Exemple avec les tailles :

min = 160
max = 190

Pour 175 :

x_scaled = ((175 - 160) / (190 - 160)) * 2 - 1
x_scaled = (15 / 30) * 2 - 1
x_scaled = 0.5 * 2 - 1
x_scaled = 0

Résultat :

Height Scaled [-1, 1]
160 -1
175 0
180 0.3333
190 1

À retenir :

La plus petite valeur devient -1.
La plus grande valeur devient 1.
La valeur du milieu devient environ 0.

3. Calcul D’un Score Avec Régression Linéaire
La régression linéaire sert à prédire une valeur numérique.

Exemple : prédire un score d’examen à partir du nombre d’heures étudiées.

Données :

Hours 1 2 3 4 5 6
Score 35 40 50 55 65 70

On cherche une droite :

Score = a * Hours + b

Avec :

a = slope, la pente
b = intercept, l’ordonnée à l’origine

Étape 1 : calculer les moyennes.

x_mean = (1 + 2 + 3 + 4 + 5 + 6) / 6 = 3.5

y_mean = (35 + 40 + 50 + 55 + 65 + 70) / 6 = 52.5

Étape 2 : calculer la pente.

Formule :

a = sum((x - x_mean) * (y - y_mean)) / sum((x - x_mean)^2)

Dans cet exemple :

a = 127.5 / 17.5
a = 7.2857

Étape 3 : calculer l’intercept.

b = y_mean - a * x_mean
b = 52.5 - 7.2857 * 3.5
b = 27

Donc le modèle est :

Score = 7.2857 * Hours + 27

Pour prédire le score après 7 heures :

Score = 7.2857 * 7 + 27
Score = 78

Réponse :

Le score prédit est 78.

À retenir :

La pente dit combien le score augmente quand x augmente de 1.
L’intercept est la valeur prédite quand x = 0.

4. K-Nearest Neighbours
KNN signifie K-Nearest Neighbours, ou k plus proches voisins.

Il sert à prédire une classe ou une valeur en regardant les points les plus proches.

Pour une classification :

On regarde les k voisins les plus proches.
Chaque voisin vote pour sa classe.
La classe majoritaire gagne.

Étapes :

  1. Choisir k.
  2. Calculer la distance entre le nouveau point et chaque point connu.
  3. Trier les distances de la plus petite à la plus grande.
  4. Prendre les k plus proches.
  5. Faire un vote majoritaire.

Distance euclidienne :

distance = sqrt((p1 - q1)^2 + (p2 - q2)^2 + ... + (pn - qn)^2)

Exemple du cours :

Instance A1 A2 A3 A4 A5 Class
X1 3 5 4 6 1 1
X2 4 6 10 3 2 2
X3 8 3 4 2 6 3
X4 2 1 4 3 6 3
X5 2 5 1 4 8 2

Nouvelle instance :

X6 = (3, 12, 4, 7, 8)

Distance avec X5 :

X5 = (2, 5, 1, 4, 8)

distance(X6, X5)
= sqrt((3-2)^2 + (12-5)^2 + (4-1)^2 + (7-4)^2 + (8-8)^2)
= sqrt(1^2 + 7^2 + 3^2 + 3^2 + 0^2)
= sqrt(1 + 49 + 9 + 9 + 0)
= sqrt(68)
= 8.25

Résumé des distances :

Voisin Distance Classe
X5 8.25 2
X1 9.95 1
X2 11.18 2
X3 11.62 3
X4 11.92 3

Avec k = 1 :

Le plus proche voisin est X5.
X5 est en classe 2.
Donc X6 est en classe 2.

Avec k = 3 :

Les 3 plus proches sont X5, X1, X2.
Classes : 2, 1, 2.

Votes :

Classe 1 : 1 vote
Classe 2 : 2 votes
Classe 3 : 0 vote

Donc :

X6 est en classe 2.

Exercices
Exercice 1 : Min-Max entre 0 et 1

Données :

[10, 20, 30, 40]

Calcule la valeur scaled de 30.

Réponse :

min = 10
max = 40
x = 30

x_scaled = (30 - 10) / (40 - 10)
x_scaled = 20 / 30
x_scaled = 0.6667

Exercice 2 : Min-Max entre -1 et 1

Données :

[10, 20, 30, 40]

Calcule la valeur scaled de 30.

Réponse :

min = 10
max = 40
x = 30

x_scaled = ((30 - 10) / (40 - 10)) * 2 - 1
x_scaled = (20 / 30) * 2 - 1
x_scaled = 1.3333 - 1
x_scaled = 0.3333

Exercice 3 : Régression Linéaire Simple

Modèle :

Score = 5 * Hours + 20

Quel score pour 8 heures ?

Réponse :

Score = 5 * 8 + 20
Score = 40 + 20
Score = 60

Exercice 4 : KNN Avec k = 1

Points connus :

Point x y Classe
A 1 1 0
B 5 5 1
C 2 2 0

Nouveau point :

P = (3, 3)

Distances :

distance(P, A) = sqrt((3-1)^2 + (3-1)^2)
= sqrt(4 + 4)
= sqrt(8)
= 2.83

distance(P, B) = sqrt((3-5)^2 + (3-5)^2)
= sqrt(4 + 4)
= sqrt(8)
= 2.83

distance(P, C) = sqrt((3-2)^2 + (3-2)^2)
= sqrt(1 + 1)
= sqrt(2)
= 1.41

Le plus proche est C.

Réponse :

Avec k = 1, P est en classe 0.

Exercice 5 : KNN Avec k = 3

P = (3, 3)

Les 3 voisins sont :

C → classe 0
A → classe 0

Votes :

Classe 0 : 2 votes
Classe 1 : 1 vote

Réponse :

Avec k = 3, P est en classe 0.
